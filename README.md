# CzAdmin-Front

The front page of cz-admin project

Stack
-----

- angular4
- rxJS
- ngrx
- ng-bootstrap
- angular/material
- angular2-datatable
- angular2-jwt   
- angular2-websocket
- nebular
- ionicons
- .....


Quick Start
-----------

```shell
$ git clone https://github.com/jomalonejia/Cz-Admin-Front.git
$ cd  Cz-Admin-Front
$ npm install
$ npm start
```

server would on localhost:4200
